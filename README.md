# Bristle 🪥

A cybersecurity hygiene reminder app built as part of CS3670 (Secure Management of Systems) at NPS.

Concept: most people know they should practice good cyber hygiene but don't think about it consistently. Bristle is a friendly nudge. A toothbrush mascot named Bristle provides random cybersecurity tips, with the goal of making security awareness a daily habit rather than an annual training checkbox.

## Current State (POC)

- Flutter Android app
- Toothbrush mascot with a greeting
- 10 cybersecurity tips, randomly served on button tap

## Planned Features

- Push notifications for scheduled reminders
- Tip categories (phishing, passwords, physical security, mobile)
- Quiz mode
- Streaks and gamification
- Backend API for tip management

## Tech Stack

- Flutter (Android)
- Node.js or Python/FastAPI backend (planned)
- PostgreSQL (planned)
- Hosted on AWS EC2 (planned)

## Running Locally

Requires Flutter SDK and Android Studio installed.

```
flutter pub get
flutter run
```
